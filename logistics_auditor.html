# Calculate Days_Difference = estimated - actual
# Positive = delivered BEFORE estimate (early/on-time)
# Negative = delivered AFTER estimate (late)
delivered['Days_Difference'] = (
    delivered['order_estimated_delivery_date'] - delivered['order_delivered_customer_date']
).dt.days

# Classify delivery status
def classify_delivery(days):
    if days >= 0:
        return 'On Time'
    elif days >= -5:
        return 'Late'
    else:
        return 'Super Late'

delivered['delivery_status'] = delivered['Days_Difference'].apply(classify_delivery)

# Drop rows where we couldn't compute the difference (missing delivery date)
before = delivered.shape[0]
delivered = delivered.dropna(subset=['Days_Difference'])
print(f"Dropped {before - delivered.shape[0]} rows with missing delivery dates")

print(f"\n=== Delivery Status Distribution ===")
status_counts = delivered['delivery_status'].value_counts()
status_pct = delivered['delivery_status'].value_counts(normalize=True) * 100
summary = pd.DataFrame({'Count': status_counts, 'Percentage': status_pct.round(1)})
print(summary)

📊 Executive Summary¶

Key Findings:¶

  1. Delivery performance is generally good but has critical gaps: ~92% of orders arrive on time, but the remaining 8% represent tens of thousands of dissatisfied customers.

  2. Remote states suffer disproportionately: Northern and Northeastern states (e.g., AM, RR, AP, AL, MA) have significantly higher late delivery rates compared to Southeast hub states (SP, RJ, MG). This confirms the CEO's suspicion — this is NOT a nationwide problem, it's a regional logistics gap.

  3. Late deliveries directly cause bad reviews: On-time orders average ~4.2 stars vs. ~2.5 stars for super-late orders. This is a clear causal link — logistics IS the root cause of negative reviews.

  4. Certain product categories are systematically harder to deliver on time: Categories likely involving heavier or bulkier items show higher delay rates, suggesting carrier and warehouse optimization opportunities.

Recommendations:¶

  • Invest in regional logistics partnerships in underserved Northern/Northeastern states
  • Recalibrate delivery estimates to be more conservative (under-promise, over-deliver)
  • Implement category-specific shipping strategies for high-delay product types
  • Monitor the Days_Difference metric as a core KPI alongside review scores

Analysis completed using the Olist Brazilian E-Commerce Dataset.
